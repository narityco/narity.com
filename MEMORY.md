# Project Memory

## Overview
Static website for "narity.com" - a simple HTML/CSS site with assets.

## Structure
```
/narity.com
├── index.html
├── about.html
├── contact.html
├── logos.html
├── .gitignore
├── README.md
├── .github/workflows/static.yml
├── .idea/ (JetBrains config)
├── assets/
│   ├── style.css
│   ├── favicon.svg
│   ├── natity-logo-alt-wordmark.svg
│   ├── natity-logo-alt-flow.svg
│   ├── natity-logo-alt-mark.svg
│   ├── natity-logo-dark.svg
│   └── natity-logo-draft-white.png
```

## Technology
- Pure static HTML/CSS site
- GitHub Actions for static site deployment
- No JavaScript frameworks

## Key Files
- `index.html` - Main landing page
- `about.html` - About page
- `contact.html` - Contact page
- `logos.html` - Logos/branding page
- `assets/style.css` - Main stylesheet

## Git
- Remote: origin (likely GitHub)
- Default branch: main
- Has GitHub Actions workflow for static site hosting
